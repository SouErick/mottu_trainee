# 🏍️ Mottu Trainee - Análise de Churn e Retenção com MottuChip

Projeto desenvolvido para o desafio técnico do programa trainee da Mottu, com foco em análise de dados, retenção de clientes e tomada de decisão baseada em métricas reais.

---

# 📌 Objetivo do Projeto

O principal objetivo deste projeto foi entender:

- o comportamento de churn dos clientes
- o impacto do **MottuChip** na retenção
- quais perfis possuem maior chance de permanência no ecossistema
- possíveis estratégias para aumentar adesão ao produto

A análise foi feita utilizando Python e técnicas de exploração, tratamento e visualização de dados.

---

# 🧠 Principais Insights

Após as análises, foi possível identificar que:

✅ Clientes da safra de 2024 que utilizam o **MottuChip** apresentam menor churn.

✅ O produto reduz em aproximadamente **30% o risco de evasão** entre clientes novos.

✅ O melhor momento para oferta do chip não é no início do contrato, mas sim após o período de adaptação do entregador.

✅ Clientes do plano **Minha Mottu** possuem maior propensão de adesão e retenção.

# 🚀 Estratégia Recomendada

Com base nos dados analisados, a recomendação principal é:

> Investir mais no MottuChip como ferramenta de retenção.

As principais estratégias sugeridas foram:

- oferta do chip no 2º mês de contrato
- foco em clientes Minha Mottu
- criação de combos com segundo chip
- integração do chip com maquininha de cartão
